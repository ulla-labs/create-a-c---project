# C++ Bubble Sort Demonstration

This is a simple C++ project that implements and demonstrates the Bubble Sort algorithm.

## Prerequisites

- GCC/G++ compiler
- (Optional) Node.js/NPM (to use the provided `package.json` scripts)

## Building and Running

### Using the provided scripts (requires NPM):

```bash
npm run build
npm run run
```

### Manual Compilation:

```bash
g++ src/main.cpp src/bubble_sort.cpp -Iinclude -o bubble_sort
./bubble_sort
```

## Project Structure

- `include/`: Header files.
- `src/`: Source code files.
- `package.json`: Build and execution scripts.
